void CSohu2::legalBrackets ( int  n)
inline

输出所有合法的括号组合

给定N对括号，输出其所有的合法的组合状态， 例如，n=3，所有的合法状态为："((()))”, “(()())”, “(())()”, “()(())”, “()()()”；
解析：深搜DFS的思路，深搜的过程关键在于记录已经用掉的左括号个数和右括号的个数， 当用过的左括号个数小于右括号则非法；当二者个数和大于2N则非法；当二者个数相等且数目等于2N则为合法。

Parameters
n输入的括号对数
Returns
NULL
Note
卡特兰数：h(n)= h(0)*h(n-1) + h(1)*h(n-2) + ... + h(n-1)h(0) (其中n>=1)，而且h（0）=1
