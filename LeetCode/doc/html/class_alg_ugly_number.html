Detailed Description

丑数是只包含因子2，3，5的数,求按照从小到大的顺序的第N个丑数。

把只包含因子2，3，5的数称为丑数(Ugly Number)，例如6和8都是丑数，但14不是，因为他包含因子7. 习惯上，我们把1当作第一个丑数。求按照从小到大的顺序的第N个丑数。
