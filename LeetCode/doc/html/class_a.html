Detailed Description

成员变量的初始化顺序只与它们在类中声明的顺序有关，与在初始化列表中的顺序无关。

Note
在C++中，成员变量的初始化顺序只与它们在类中声明的顺序有关，而与在初始化列表中的顺序无关。 此题中,n1先于n2声明，因此n1也会在n2之前初始化，所以先用n2+2初始化n1，此时n2的值是随机的， 故n1也是一个随机值，然后用0初始化n2.
